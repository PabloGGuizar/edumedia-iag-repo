Filtrado por URL

Puedes compartir un enlace a esta página con filtros ya aplicados añadiendo parámetros a la URL. La búsqueda no distingue entre mayúsculas/minúsculas ni acentos.

Ejemplos:

  • Para filtrar por un autor:
    ?autor=Maria Lopez
  • Para filtrar por palabra clave:
    ?keyword=juego
  • Para filtrar por título:
    ?titulo=sudoku
  • Para filtrar por URL de la app:
    ?url=github.io
  • Para filtrar por área:
    ?area=matematicas
  • Para filtrar por varios niveles (separados por comas):
    ?nivel=Primaria,Secundaria
